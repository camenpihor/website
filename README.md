# Website

## Install

1. `npm install` to install frontend
2. Install backend
   1. Follow instructions at `backend/README.md`

## Run Servers

In separate processes:

1. development flow
   1. `scripts/run-frontend`
   2. `cd backend`
   3. `scripts/run-backend --dev`
